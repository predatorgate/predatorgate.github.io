# Predatorgate Timeline

Η σελίδα αυτή παρουσιάζει ένα ιστορικό του σκανδάλου παρακολουθήσεων Predatorgate, που έλαβε χώρα στην Ελλάδα κατά τη διακυβέρνηση Κυριάκου Μητσοτάκη.
